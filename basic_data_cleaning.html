Missing Data Cleaning

We see from above that we are missing a lot of data from the columns nedbør, snødekningsgrad, snøvannekvivalent, temperatur, and MVF. To view the missing data visually, we use the package naniar:

library(naniar)
vis_miss(eggafoss)

It seems like all the missing data is from the first 6119 rows. This can be verified by the command:

any(is.na(eggafoss[6120:28764, ]))
## [1] FALSE

For the first 6119 rows, we are missing so much data that the best option is to remove the first 6119 rows. This corresponds to data before 1958.

eggafoss_from_1958 <- eggafoss[6120:28764, ]

To get a visual look at the correlation matrix, we use the packages reshape2 and ggplot2:

library(reshape2)
library(ggplot2)

ggplot(data = melt(correlation_matrix), aes(x = Var1, y = Var2, fill = value)) +
  geom_tile()

Here we can see some predictable relations, like temperature being highly negatively correlated with snødekningsgrad. It seems like vannstand is highly positively correlated with all the four variables temeratur, snøvannekvivalent, snødekningsgrad, and nedbør.

Scatterplot

To avoid to many points at once, we sample 2000 points from our data so that the underlying structure of the data becomes clearer:

eggafoss_sample <- eggafoss_from_1958[sample(nrow(eggafoss_from_1958), 2000), ]

We now some basic scatterplots with ggplot2:

library(ggplot2)
ggplot(data = eggafoss_sample, aes(x = temperatur, y = vannstand)) + geom_point() +
  geom_smooth()
## `geom_smooth()` using method = 'gam' and formula 'y ~ s(x, bs = "cs")'

Here we see that there is a positive trend, that is most clear in the temperature interval between -10 and 5. This can indicate that vannstand is seasonal-dependend, and that a high vannstand occurs in the spring.

Below, we do three more scatterplots illustrating how vannstand depends on the other variables it is highly positively correlated with:
